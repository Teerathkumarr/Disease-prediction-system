What We Detect

Two Modules.
One Unified System.

Each module uses a different combination of modalities — optimized for the specific disease's clinical indicators and available data sources.

🫀

Cardiovascular Disease

Analyzes ECG signals using a 1D-CNN + LSTM model to detect arrhythmias and abnormal heart patterns, fused with clinical biomarkers like cholesterol, blood pressure, and age using XGBoost.

1D-CNN + LSTM ECG Signal XGBoost Late Fusion SHAP PTB-XL Dataset
🩸

Diabetes Detection

Detects diabetic retinopathy from fundus images using EfficientNet-B4, fused with lab values like HbA1c, glucose, BMI, and insulin via a trained tabular model.

EfficientNet-B4 Retinal Fundus XGBoost Grad-CAM SHAP APTOS Dataset
🔬

Explainability Layer

Every prediction comes with visual and quantitative explanations — Grad-CAM heatmaps highlight suspicious regions in images, while SHAP waterfall plots rank feature importance for tabular data.

Grad-CAM SHAP Captum Attention Maps
📊

Risk Reporting

Generates a detailed PDF clinical report with risk score (0–100%), severity level, SHAP breakdown, and actionable recommendations — ready to print for the attending physician.

Layer 01 · Data

Data Engineering

Multi-source data collection from NIH, PhysioNet, Kaggle. Preprocessing includes normalization, augmentation (rotation, flipping, CLAHE for images), and SMOTE for class balancing.

Pandas OpenCV Albumentations imbalanced-learn
Layer 02 · Models

Deep Learning Models

EfficientNet-B4 for retinal images (pretrained ImageNet). 1D-CNN + BiLSTM for ECG time-series. XGBoost with hyperparameter tuning via Optuna for tabular features.

PyTorch EfficientNet-B4 BiLSTM XGBoost Optuna
Layer 03 · Fusion

Multimodal Fusion

Late fusion architecture: each modality produces a probability vector, concatenated and passed through a 2-layer MLP for final prediction. Confidence via MC Dropout.

Late Fusion MLP MC Dropout Softmax
Layer 04 · XAI

Explainability

Grad-CAM generates class activation maps on retinal/ECG images. SHAP TreeExplainer for XGBoost. Captum for PyTorch models. Every prediction is fully interpretable.

Grad-CAM SHAP Captum LIME
Layer 05 · Backend

API & Backend

FastAPI REST backend with async inference endpoints. JWT authentication for doctor accounts. PostgreSQL database for patient records. Redis for caching model outputs.

FastAPI PostgreSQL Redis JWT Auth SQLAlchemy
Layer 06 · Deployment

Frontend & Deploy

React.js dashboard with Chart.js visualizations. PDF report generation with ReportLab. Dockerized containers deployed on AWS EC2. CI/CD via GitHub Actions.

React.js Docker AWS EC2 ReportLab GitHub Actions
